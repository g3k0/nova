# Nova

Nova is a personal author website project designed to publish short stories and literary works with a strong visual identity.

## Vision
Create a digital literary space where writing quality and reading experience are equally important, combining elegant design with a simple and sustainable publishing workflow.

## Mission
- Publish and curate original literary content in a professional way.
- Offer a clean, immersive, and accessible reading experience.
- Keep full control of content and platform through an open, self-hosted architecture.

## Project Status
This project is in an early definition phase.  
Architecture and implementation planning are in progress, and details may evolve over time.

## Technology Stack
The project currently targets a hybrid architecture:

- **Frontend:** Next.js (React + TypeScript)
- **CMS:** Strapi Community Edition (self-hosted)
- **Database:** PostgreSQL (preferred for production scenarios)
- **Infrastructure:** self-hosted deployment with container-friendly setup (Docker-based workflow)

Why this stack:
- high-quality custom UI/UX on the frontend;
- editorial autonomy via headless CMS;
- zero software licensing cost with full technical control.

## Repository Structure (Initial)

## Guiding Principles
- **Content first:** writing and storytelling are the core product.
- **Design matters:** visual quality is part of the reading experience.
- **Simplicity over complexity:** prioritize a focused MVP before advanced features.
- **Ownership:** prefer open tools and self-hosting to reduce platform lock-in.
